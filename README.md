# 🧠 Online Exam System (JavaFX + MySQL)

## 📌 Overview
A desktop-based Online Examination System built using JavaFX and MySQL.  
It supports MCQ, True/False, and Text-based questions with real-time timer and proctoring features.

---

## ✨ Key Features
- Student login system
- Dynamic question loading from MySQL
- MCQ / True-False / Text questions
- Exam timer with auto-submit
- Randomized questions
- Auto scoring system
- Result generation (Pass/Fail)
- Review answers feature
- Basic proctoring (tab switch + warnings + face detection simulation)
- Admin Module with Dashboard

---

## 🛠 Tech Stack
- Java
- JavaFX
- MySQL
- JDBC

---

## 🗄 Database Tables
- users
- questionss
- exam_settings
- results

---

## ⚙️ How to Run
1. Clone repository
2. Open in IntelliJ / Eclipse
3. Setup MySQL database
4. Update DB credentials in code
5. Run `ExamPage.java`
